# 📊 Restaurant Analytics Dashboard

## 🚀 Overview
End-to-end data analytics project analyzing **11M+ restaurant records** from CSV & JSON sources.  
Built using **Databricks** for data processing and **Power BI** for interactive visualization.

---

## ⚙️ Tools
- Databricks (SQL)
- Power BI
- DirectQuery
- CSV & JSON

---

## 📊 Key Metrics
- 💰 Revenue: ~3B  
- 🧾 Orders: 11M  
- 📍 Top Branch: Cairo  
- 🕗 Peak Hour: 8 PM  
- ⭐ Avg Rating: 3.7  

---

## 📈 Dashboard Pages
- **Overview**: KPIs & trends  
- **Sales**: Top items & categories  
- **Behavior**: Customer patterns & peak times
